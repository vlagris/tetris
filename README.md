# Tetris

<p>
    <img alt="NPM Badge" src="https://img.shields.io/badge/v10.2.3-green?label=npm&color=blue">
</p>

<p>
    Веб версия игры Tetris сделанная на React
</p>


## Tech Stack
- TypeScript
- React
- SASS
- Vite


## Quick Start

#### Cloning the Repository
```
git clone git@github.com:vlagris/tetris-app.git
cd tetris-app
```

#### Installation
```
npm install
```

#### Running the Project
```
npm run dev
```
